# KMeans
The project implemented the k-means algorithm and some of its variants, including
* k-means and its concurrent version
* k-means++ <sup>[1]</sup>
* k-means|| <sup>[2]</sup>
* k-MC<sup>2</sup> <sup>[3]</sup>
